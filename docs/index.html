LOGOVO — Кафе-бар

Website for a trendy cafe-bar with menu and reservation system

📋 Project Overview

A modern, fully responsive website for LOGOVO cafe-bar featuring:

✓ Interactive menu browser
✓ Table reservation system
✓ Photo gallery
✓ Contact information
✓ Operating hours
✓ Responsive design

🛠️ Technology Stack

Built with modern web technologies:

HTML5 CSS3 JavaScript (ES6+) Responsive Design Google Fonts

Language Composition

HTML: 59.2%
CSS: 33.0%
JavaScript: 7.8%

✨ Key Features

Menu System

Full menu with categorized items (appetizers, salads, main courses, desserts, beverages, bar). Each item includes name, description, price, and portion weight.

Booking System

Easy-to-use reservation form for customers to book tables at the cafe.

Responsive Design

Optimized for all devices - mobile phones, tablets, and desktop computers.

Modern UI

Beautiful gradient backgrounds, smooth animations, and elegant typography using Google Fonts (Playfair Display and Inter).

📁 Project Structure

logovo-cafe/
├── index.html    — Main HTML page
├── styles.css    — Stylesheet
├── script.js     — Navigation, menu tabs, form logic
├── docs/         — GitHub Pages documentation
└── README.md     — Project documentation

🚀 Getting Started

To run the project locally, open index.html in your browser or use an HTTP server:

# Python 3
python3 -m http.server 8080

# Node.js
npx serve .

Then visit http://localhost:8080 in your browser.
